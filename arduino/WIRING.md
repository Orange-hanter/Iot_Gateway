# Схема подключения MQ2 к Arduino Mega

## Визуальная схема

```
                    ┌─────────────────────────┐
                    │   Arduino Mega 2560     │
                    │                         │
                    │                      5V ├─────┐
                    │                     GND ├───┐ │
                    │                      A0 ├─┐ │ │
                    │                Pin 2 (D2)├ │ │ │
                    │                  LED (13)│ │ │ │
                    │                         │ │ │ │
                    └─────────────────────────┘ │ │ │
                                                │ │ │
                    ┌───────────────────────────┼─┼─┼─┐
                    │       MQ2 Gas Sensor      │ │ │ │
                    │                           │ │ │ │
                    │  ┌───┐                    │ │ │ │
                    │  │ ◉ │ ← Heating Element  │ │ │ │
                    │  └───┘                    │ │ │ │
                    │                           │ │ │ │
                    │  VCC ─────────────────────┘ │ │ │
                    │  GND ───────────────────────┘ │ │
                    │   A0 ─────────────────────────┘ │
                    │   D0 ───────────────────────────┘
                    │                           │
                    └───────────────────────────┘
```

## Таблица подключений

| MQ2 Pin | Назначение          | Arduino Mega Pin | Описание                              |
|---------|---------------------|------------------|---------------------------------------|
| VCC     | Питание             | 5V               | Питание 5 вольт                      |
| GND     | Земля               | GND              | Общая земля                          |
| A0      | Аналоговый выход    | A0               | Аналоговое значение (0-1023)         |
| D0      | Цифровой выход      | Pin 2 (D2)       | HIGH при превышении порога           |

## Физическое расположение пинов MQ2

```
    ┌───────────────┐
    │  MQ2 Sensor   │
    │   [Front]     │
    │               │
    │   ┌───────┐   │
    │   │   ◉   │   │ ← Нагревательный элемент
    │   └───────┘   │
    │               │
    └─┬─┬─┬─┬─┬─┬──┘
      │ │ │ │ │ │
      A G V D N N
      0 N C O C C
        D C

Pin 1: A0  - Analog Out
Pin 2: GND - Ground
Pin 3: VCC - Power (5V)
Pin 4: D0  - Digital Out
Pin 5: NC  - Not Connected
Pin 6: NC  - Not Connected
```

## Расположение пинов на Arduino Mega

```
Arduino Mega 2560 - Digital Pins (правая сторона)

    ┌─────────────────────┐
    │ USB ████████         │
    │                      │
    │  Power               │
    │  [RESET] ●           │
    │  [ 3V3 ]             │
    │  [  5V ] ●───────────┼─→ К MQ2 VCC
    │  [ GND ] ●───────────┼─→ К MQ2 GND
    │  [ GND ]             │
    │  [ Vin ]             │
    │                      │
    │  Analog              │
    │  [  A0 ] ●───────────┼─→ К MQ2 A0
    │  [  A1 ]             │
    │  [  A2 ]             │
    │   ...                │
    │                      │
    │  Digital             │
    │  [   0 ]             │
    │  [   1 ]             │
    │  [   2 ] ●───────────┼─→ К MQ2 D0
    │  [   3 ]             │
    │   ...                │
    │  [  13 ] ● (LED)     │ ← Встроенный LED
    │                      │
    └─────────────────────┘
```

## Важные замечания

1. **Полярность**: Соблюдайте полярность! VCC → 5V, GND → GND
2. **Напряжение**: MQ2 работает от 5V (не подключайте к 3.3V)
3. **Потребление**: ~150mA при прогреве (Arduino может обеспечить до 500mA на 5V)
4. **Прогрев**: Датчик требует 20-30 секунд прогрева после включения
5. **Калибровка**: Для точности откалибруйте в чистом воздухе минимум 24 часа

## Дополнительные подключения (опционально)

### LED индикатор (встроенный)
- Уже используется в прошивке на Pin 13
- Мигает при отправке данных

### Внешний LED индикатор тревоги
Если хотите добавить внешний LED:

```
Pin 3 (PWM) → [Резистор 220Ω] → [LED +] → [LED -] → GND
```

### Buzzer (пищалка) для звуковой сигнализации

```
Pin 4 → [+Buzzer] → [-Buzzer] → GND
```

## Проверка подключения

После подключения:

1. Подключите Arduino к USB
2. Откройте Serial Monitor (115200 baud)
3. Должны увидеть JSON сообщения
4. LED на Pin 13 должен мигать каждые 2 секунды

## Устранение проблем

| Проблема                  | Возможная причина          | Решение                          |
|--------------------------|----------------------------|----------------------------------|
| Нет данных в Serial      | Неверная прошивка          | Перепрошейте Arduino            |
| Значения = 0             | Датчик не подключен        | Проверьте соединения            |
| Значения = 1023          | Короткое замыкание         | Проверьте VCC/GND               |
| Нестабильные показания   | Недостаточный прогрев      | Подождите 30 мин - 24 часа      |
| Arduino не определяется  | Драйвер USB                | Установите CH340 драйвер        |
