## Принципы

1. **Декомпозиция** — один файл = одна тема, без огромных файлов
2. **Ссылочная модель** — документы ссылаются друг на друга, не дублируют
3. **Маркер `<!-- @TODO -->`** — для мест требующих проработки
4. **Verbose** — лучше описать подробно, чем упустить контекст
5. **Obsidian-совместимость** — wiki-ссылки `[[файл]]` и теги `#tag`
